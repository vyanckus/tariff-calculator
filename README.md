# 🚚 Тарифный калькулятор для сервиса доставки грузов

Микросервис для расчета стоимости доставки грузов с поддержкой:
- расчета по весу и объему груза
- расчета расстояния между географическими координатами
- гибкой конфигурации тарифов

## 🎯 О проекте

Данный проект был разработан в рамках учебной программы по Java-разработке.
Представляет собой готовый к использованию микросервис, который может быть
интегрирован в систему интернет-магазина или логистической компании для
мгновенного расчета стоимости доставки.

### Ключевые возможности

- ✅ **Расчет по весу** - базовая стоимость за килограмм
- ✅ **Расчет по объему** - стоимость за кубический метр с округлением габаритов
- ✅ **Расчет расстояния** - по географическим координатам (формула гаверсинусов)
- ✅ **Гибкая настройка** - все тарифы вынесены в конфигурационный файл
- ✅ **Валидация входных данных** - проверка веса, габаритов и координат
- ✅ **Обратная совместимость** - поддержка старых версий API

## 🏗️ Архитектура

Проект построен на принципах **Clean Architecture**:

app (запуск и конфигурация)  
↑  
web (контроллеры, DTO)  
↑  
useCase (бизнес-сценарии)  
↑  
domain (бизнес-объекты и правила)


- **domain** - независимое ядро с бизнес-логикой
- **useCase** - сценарии использования
- **web** - REST API контроллеры
- **app** - сборка и конфигурация Spring

## 📦 API эндпоинты

### POST `/api/v1/calculate`

**Пример запроса (полный):**

```json
{
  "packages": [
    {
      "weight": 4564,
      "length": 345,
      "width": 589,
      "height": 234
    }
  ],
  "currencyCode": "RUB",
  "departure": {
    "latitude": 55.446008,
    "longitude": 65.339151
  },
  "destination": {
    "latitude": 62.035450,
    "longitude": 55.027532
  }
}
```

Пример ответа:

```json
{
  "totalPrice": 3815.51,
  "minimalPrice": 350,
  "currencyCode": "RUB"
}
```

### Алгоритм расчета

1.  **Нормализация** - округление габаритов кратно 50 мм вверх

2.  **Расчет по весу** - вес в кг × цена за кг

3.  **Расчет по объему** - объем в м³ × цена за м³

4.  **Базовая стоимость** - max(вес, объем, минимальная цена)

5.  **Применение коэффициента расстояния** - баз. стоимость × (расстояние / 450 км)


### Валидация

| Параметр | Ограничение |
|----------|-------------|
| Вес упаковки | ≤ 150 кг |
| Длина/ширина/высота | 0-1500 мм |
| Широта | 45° - 65° |
| Долгота | 30° - 96° |

## ⚙️ Конфигурация

Все тарифы настраиваются в `application.yml` без перекомпиляции:

```yaml
cost:
  rub:
    perKg: 400           # цена за кг
    minimal: 350         # минимальная стоимость
    volume:
      perCubicMeter: 5000 # цена за м³

coordinates:
  latitude:
    min: 45
    max: 65
  longitude:
    min: 30
    max: 96
```

## 🚀 Запуск

## 🔒 Ключевые бизнес-правила

*   **Минимальная стоимость** - 350 рублей

*   **Максимальный вес упаковки** - 150 кг

*   **Минимальное расстояние для коэффициента** - 450 км

*   **Округление цены** - до 1 копейки в большую сторону

*   **Округление габаритов** - до 50 мм вверх
